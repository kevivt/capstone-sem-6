# ISA Review 2 Narrative Plan

Audience: capstone review panel and guide.

Objective: keep the same slide headings and overall sequence as ISA Review 1, but update the content to reflect the current Phase 2 implementation status.

1. Restate the problem and scope.
2. Show how Phase 1 feedback changed the implementation path.
3. Document the dataset amalgamation and identify which sources are active for serving.
